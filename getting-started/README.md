# Getting started

Complete project implementation from tutorial [Getting Started with Knot.x Stack](http://knotx.io/tutorials/getting-started-with-knotx-stack/)

## Run
Build first
```
$ gradlew clean build
```

Unzip `build/distribution/knotx-stack-version.zip`

Run

```
$ bin/knotx run-knotx
```

###Final page

http://localhost:8092/content/books.html
